**THINGS TO CHECK IN ANY LINUX DISTRO**

>Kernel version

```
uname -r
```

>Release version

```
cat /etc/*release
```

>KDE version

```
plasmashell --version
```

>Gnome version

```
gnome-shell --version
```


>Details can be looked up with `neofetch` or `fastfetch`, depending if they are installed.
