**XFCE PANELS TLDR**

The best way to export XFCE4 panel configurations is using xfce4-panel-profiles, the official tool designed for managing and transferring panel layouts.

>Save current panel layout to a file

```
xfce4-panel-profiles save my-panel-config.tar.bz2
```

>Load and apply a saved configuration

```
xfce4-panel-profiles load my-panel-config.tar.bz2
```
